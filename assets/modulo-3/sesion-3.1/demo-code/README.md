# Demo Code — Sesión 3.1: De Boceto a Producto con Claude Vision

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `wireframe.excalidraw` | Wireframe dibujado en Excalidraw que Claude Vision convierte a código |

## Flujo de la demo

1. Abrir `wireframe.excalidraw` en [Excalidraw](https://excalidraw.com)
2. Exportar como PNG
3. Enviar a Claude Code con el prompt:
   ```
   Convierte esta imagen de wireframe en una Landing Page
   completa con Next.js 15 + Tailwind CSS.
   Usa colores primary #1e3a5f y accent #0ea5e9.
   ```
4. Ver el resultado en `app/page.tsx`
