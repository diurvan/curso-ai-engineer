# Recursos — Sesión 1.2: Tu Primer Agente Productivo

## Paquete @curso-ai/metrics
- [Repositorio del curso — `packages/metrics/`](/packages/metrics/)
- `trackAnthropic()` — Extrae usage del response de Anthropic y reporta al dashboard
- `trackOpenAI()` — Extrae usage del response de OpenAI y reporta al dashboard
- `syncHelicone()` — Sincroniza métricas de Helicone al dashboard
- `estimateCost()` — Calcula costo según modelo y tokens
- `report()` — Reporte manual (fallback para proveedores no cubiertos)
